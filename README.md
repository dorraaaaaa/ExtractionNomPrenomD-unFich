L’objectif :
Lire un fichier texte encodé en UTF-16-LE 
Analyser le texte pour y détecter des entités de type Prénom + Nom,
Filtrer correctement les résultats (ne pas accepter n’importe quels mots).
Enregistrer les résultats dans un fichier de sortie encodé en UTF-8 avec BOM.

Ce travail se base principalement sur la gestion des encodages et l’extraction simple d’information dans un texte, sans utiliser de bibliothèques avancées.
